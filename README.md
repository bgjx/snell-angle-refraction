# What this code about ?
> A cheap piece of code to calculate the take-off angle, travel time and incidence angle at station using shooting snell's method in local flat layered study.

## Tools
```
- Python 3.10
- Numpy 1.23
- Matplotlib 3.5
- ObsPy 1.4.1
```
